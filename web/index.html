STOCK

Inicialmente denominado ICJ (Index de Computadores de Jaboatão), o projeto tem como propósito criar um catálogo para os computadores da Secretaria de Saúde da prefeitura, permitindo o monitoramento remoto desses ativos. O objetivo é melhorar o controle do patrimônio público e a eficácia na tomada de decisões. Ao possibilitar o acompanhamento remoto, o projeto visa identificar e resolver problemas técnicos rapidamente. Para mais detalhes, a documentação está disponível para download.
